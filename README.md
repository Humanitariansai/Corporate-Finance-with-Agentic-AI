# **Course Overview**

**Corporate Finance with Agentic AI** reimagines the traditional corporate finance curriculum through the lens of:

* **Python-based financial modeling and simulation**,
* **LLMs as reasoning and analytical partners**, and
* **Agentic AI systems that plan, analyze, and validate financial decisions**.

Volunteers build:

* Automated valuation models
* RL + LLM financial agents
* Governance and risk analysis agents
* NLP pipelines for financial documents
* End-to-end agentic systems for capital budgeting, forecasting, and M&A

The course uses the Humanitarians AI **Learn → Build → Teach** pedagogy:

1. Begin with an outline
2. Build labs before videos
3. Use Humanitarians Courses GPT for scripts and slides
4. Record YouTube mini-lectures
5. Iterate through GitHub and volunteer updates
